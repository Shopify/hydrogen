# Hydrogen

Shopify storefront SDK, React examples, and supporting tooling.

## Repository Layout

- `packages/` — publishable packages.
- `examples/` — React examples and shared example helpers.
- `patches/` — pnpm patches required by the workspace.

## Common Commands

- `pnpm install`
- `pnpm run build`
- `pnpm run typecheck`
- `pnpm run test`
- `pnpm run lint`
- `pnpm run format:check`
- `pnpm run check`

## Package

The main package currently lives at `packages/hydrogen` and publishes as `@shopify/hydrogen`.

## Examples

See `examples/README.md` for the React examples and how to run them.
